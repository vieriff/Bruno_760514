++REQUISITI / REQUIREMENTS++

L'applicazione richiede Java JDK 21 e sistema operativo Windows 11.  
Nota: il programma è stato sviluppato e testato su Windows 11. Il funzionamento su altri sistemi operativi (Linux, macOS) non è garantito.

The application requires Java JDK 21 and Windows 11 OS.  
Note: the program was developed and tested on Windows 11. Functionality on other operating systems (Linux, macOS) is not guaranteed.

++COME AVVIARE L’APPLICAZIONE / HOW TO RUN THE PROGRAM++

All’interno della cartella `/bin` è presente il file `start_theknife.bat`.  
Eseguendolo con doppio clic, si aprirà una finestra del prompt dei comandi da cui sarà possibile utilizzare l'applicazione.

Inside the `/bin` folder, you will find the `start_theknife.bat` file.  
Double-clicking it will open a command prompt window where you can use the application.

++ERRORI CONOSCIUTI / KNOWN ISSUES++

- Il programma è stato testato più volte e non presenta attualmente errori noti.  
- Tuttavia, **se i file eseguibili vengono spostati dalla posizione originale (cartella /bin), il programma potrebbe non avviarsi correttamente**, poiché si basano su percorsi relativi ai file interni.

- The program has been tested several times and currently shows no known issues.  
- However, **if the executable files are moved from their original location (/bin folder), the program may not start correctly**, as it relies on relative paths to access internal files.
