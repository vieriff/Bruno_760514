Come avviare / How to run the program

- Nella cartella /bin si trova il file start_theknife.bat. Avviandolo, si aprirà una finestra del prompt dei comandi in cui sarà possibile utilizzare il programma.

- Inside the /bin folder, you will find the start_theknife.bat file. Running this file will open a command prompt window where you can use the program.
